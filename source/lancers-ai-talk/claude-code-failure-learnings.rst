cdcasasagi（鵲）
======================================================================

* HTTP StreamableなMCPサーバをmcp-proxyでstdioに見せる特殊な場合
* Claude Desktopの設定ファイル :file:`claude_desktop_config.json` を簡単に生成
* Windows・macOS両対応のPython製CLIツール

cdcasasagiの開発
======================================================================

:実装: Opus
:レビュー: GPT

実装はClaude
--------------------------------------------------

* 半分以上は通勤中に開発
* webでplanし、実装後、プルリクエストの面倒を（人力交え）見させる
* ローカルで開発した後、 ``/autofix-pr`` でwebに面倒見させる

レビューはGPT
--------------------------------------------------

* Claude CodeプラグインでStop hookでCodexレビュー発火
* GitHubのPRへのpushをトリガーにCodex cloudがレビューするように設定

私に **簡単** だったのでこの形
--------------------------------------------------

* Claude ``/autofix-pr``
* GPT レビュー設定が簡単（特にcloud）

.. ``/loop`` ``/goal``

実装をClaudeに任せてみて
======================================================================

* Opus単体には任せられないなあ
* 序列：Opus < 私 < GPT

※今回のPython製小さなCLIに限った話（私の使いこなし不足もあるかと）
